# Project Bible

## Product
Cloud-based Shirt Manufacturing ERP SaaS.

## Tech Stack
- React + Vite + TypeScript
- Tailwind CSS
- Node.js + Express
- PostgreSQL
- Prisma ORM
- JWT Authentication

## Core Principles
- Multi-tenant
- Secure
- Modular
- API-first
- Production-ready

## Main Modules
Authentication, Dashboard, Customers, Suppliers, Products, Fabric,
Inventory, Purchase, Sales, Production, Cutting, Stitching,
Finishing, Packing, Dispatch, Invoice, Payments, Accounts,
Reports, Settings.

## Definition of Done
- Business logic complete
- Validation
- Permissions
- Tests
- Responsive UI
- Documentation
